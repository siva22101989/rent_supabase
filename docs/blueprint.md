# **App Name**: Grain Flow

## Core Features:

### Landing Page & Public Access

- **Professional Landing Page**: Modern, responsive landing page with Hero, Features, About, and CTA sections
- **Adaptive Theming**: Full light/dark mode support with seamless theme switching
- **Contact Integration**: Interactive dialog for sales contact via phone (+91 9160606633) or WhatsApp
- **Pricing Page**: Transparent pricing with Free, Starter, Professional, and Enterprise tiers
- **Smooth Navigation**: Section-based navigation with smooth scrolling

### Warehouse Management

- **New Storage Entry**: Create storage records with customer, commodity, quantity, and date-in details
- **Automated Rent Calculation**: Calculate initial and recurring rent (₹36/bag for 6 months, ₹55/bag for 1 year)
- **Active Storage Dashboard**: Display active storage records with customer name, bags stored, billing dates, and rates
- **Rollover/Renewal Alerts**: Automated alerts for storage due for rollover (6 months) or renewal (1 year)
- **Goods Withdrawal**: Process withdrawals by setting StorageEndDate with withdrawal notices
- **Invoice Generation**: Auto-generate invoices for billing, rollovers, and renewals

### Customer Management

- **Customer Portal**: Self-service portal for customers to view their storage and payments
- **Customer Profiles**: 360-degree view with active stock, history, and dues
- **Activity Timeline**: Full history of inflows, payments, and withdrawals

## Style Guidelines:

### Colors

- **Primary**: Sky Blue (#0EA5E9) - Trust, flow, technology
- **Secondary**: Amber (#F59E0B) - Grain, warmth, agriculture
- **Background (Light)**: White (#FFFFFF) / Slate 50 (#F8FAFC)
- **Background (Dark)**: Slate 950 (#020617) / Slate 900 (#0F172A)
- **Accent**: Primary with hover states

### Typography

- **Font Family**: Inter (sans-serif) - Modern, clean, highly readable
- **Headings**: Bold (700), tracking-tight
- **Body**: Regular (400), line-height-relaxed
- **UI Elements**: Medium (500)

### Design Principles

- **Clean Layout**: Structured sections with clear visual hierarchy
- **Modern Icons**: Lucide React icons for consistency
- **Smooth Animations**: Subtle transitions (200-300ms duration)
- **Responsive First**: Mobile-optimized with desktop enhancements
- **High Contrast**: WCAG AA compliant in both light and dark modes
- **Component Library**: Shadcn UI for consistent, accessible components

### UI Patterns

- **Cards**: Rounded borders (0.5rem), subtle shadows, hover states
- **Buttons**: Primary (filled), Secondary (outlined), Ghost (text only)
- **Dialogs**: Centered, backdrop blur, smooth entrance animations
- **Forms**: Inline validation, clear error states, optimistic updates
- **Navigation**: Sticky header, smooth scroll, active section highlighting
