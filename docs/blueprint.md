# **App Name**: BagBill

## Core Features:

- New Storage Entry: Create a new storage record with customer, commodity, quantity, and date-in details.
- Automated Rent Calculation: Automatically calculate the initial and recurring rent based on the defined policy (₹36/bag for 6 months, ₹55/bag for 1 year).
- Active Storage Dashboard: Display all active storage records with key information like customer name, bags stored, start date, next billing date, and current rate.
- Rollover/Renewal Alerts: Provide automated alerts when storage records are due for rollover (after 6 months) or renewal (after 1 year).
- Goods Withdrawal: Process goods withdrawal by setting the StorageEndDate and providing withdrawal notices regarding refunds.
- Invoice Generation: Automatically generate invoices for initial billing, rollovers, and renewals, and email them to customers.
- Anomaly Detection Tool: Use a generative AI tool to identify unusual patterns or anomalies in storage durations or quantities, suggesting potential errors or contract violations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light blue-gray (#E8EAF6), providing a neutral backdrop.
- Accent color: Yellow-orange (#FFB300) for calls to action and highlighting important information.
- Body and headline font: 'Inter', a sans-serif font, providing a modern, machined, objective, neutral look; suitable for headlines or body text
- Use simple, clear icons to represent actions and data points, ensuring easy navigation and quick understanding.
- Implement a clean and structured layout with clear sections for data input, active storage overview, and billing information.
- Use subtle animations for transitions and feedback, such as loading indicators and confirmation messages.
